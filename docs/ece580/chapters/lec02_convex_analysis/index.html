    Convex Analysis #

    Here is our cost function

    $$ \underset{x}{min} ||A\cdot X - B||_2^2 $$

    This is a quadratic function of \(X\) and is always non-negative.

    We can use this property to prove its convexity.

    Convex Function #

    \(\text{If a quadratic function is always non-negative}\\\ \text{its quadratic coefficient matrix is positive semi-definite ...(1)}\) \(\text{A quadratic function is convex,}\\\ \text{if and only if its coefficient matrix is positive semi-definite...(2)}\)

    Because of (1) and (2), our cost function is a convex function

    Other properties #

    Even if we have (1), it does not mean if our quadratic coefficient matrix is positive semi-definite, then the function is non-positive.

    A sufficient but not necessary condition for convexity is the Hessian matrix is positive semi-definite for all X.

    Minimizing a convex function is much easier than a general nonlinear function.

    Constraints and Feasibility #

    When facing some practical problems, we need to give some constraints, which can be expressed as:

    $$ g(x) = 0 $$

    A point X is feasible if it satisfies:

    $$ g_1(x) \leq 0, g_2(x) \leq0… $$

    The set of all feasible points is called the feasible set or constraint set.

    Convex Set #

    $$ \text{A set D is convex, if for all }X_1, X_2 \in D \text{ and 0} \leq \alpha \leq 1, \text{we have} \\ \alpha X_1 + (1-\alpha) X_2 \in D $$

    Properties #

    \[\alpha\text{-sublevel set is defined as:} \\\ \{ X|f(X)\leq \alpha \}\]
    • If f(X) is convex, its sublevel sets are convex.
    • Set convexity is preserved under intersection.

    Convex Optimization #

    If all constraints \(g_i(X)\) are convex, the constraint set is convex.

    Because the constraint set is the intersection of all convex 0-sublevel sets.

    The minimization of a convex cost function over a convex constraint set is called convex optimization.
